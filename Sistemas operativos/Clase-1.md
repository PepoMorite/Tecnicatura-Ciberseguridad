# Sistemas Operativos — Clase 1

## ¿Por qué existen los sistemas operativos?

Las computadoras modernas tienen muchos componentes: procesador, memoria, discos, teclado, red, etc.

Sería muy difícil para los programadores manejar directamente todos esos elementos del hardware.

Por eso existe el sistema operativo, que es una capa de software que:

- simplifica el uso de la computadora
- administra los recursos del hardware
- ofrece una interfaz más simple para usuarios y programas

## El sistema operativo como máquina extendida

El sistema operativo crea **abstracciones** que simplifican el uso del hardware.

Por ejemplo, en lugar de trabajar directamente con un disco duro complejo, el sistema operativo lo presenta como una colección de **archivos** que se pueden:

- abrir
- leer
- escribir
- cerrar

Esto permite que los programadores trabajen de forma mucho más simple.

## Cliente

Un cliente es una entidad que usa un o varios servicios que proveen otras entidades o entidad

En el caso del sistema operativo, los clientes son:

- los programas que se ejecutan en la computadora
- los programadores que desarrollan esos programas
- el usuario que utiliza la computadora

## El sistema operativo como administrador de recursos

El sistema operativo también administra los recursos del hardware.

Se encarga de:

- asignar procesador
- administrar memoria
- controlar dispositivos de entrada/salida

Además permite que varios programas se ejecuten al mismo tiempo sin interferir entre sí.
